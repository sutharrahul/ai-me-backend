## Featured Project: AI Portfolio Chatbot

This is my most advanced project to date: an interactive AI chatbot
version of this portfolio. Instead of browsing static pages, visitors
can ask questions naturally about my background, skills, projects, and
career, and get answers grounded in my own knowledge base using
Retrieval-Augmented Generation (RAG) rather than generic AI responses.

I built it to demonstrate production-ready LLM application architecture
and to explore streaming responses, conversation memory, and semantic
search hands-on. It combines full-stack web development with modern AI
techniques: the backend classifies each message's intent so only
portfolio-related questions trigger document retrieval (making
unrelated chit-chat faster and cheaper), retrieves relevant chunks from
a Qdrant vector database for grounded answers, and streams the response
token-by-token for a smooth, ChatGPT-like experience. Conversations
persist across reloads, are automatically titled, and are internally
split into storage chunks so long chats don't hit database or context
limits while still appearing as a single conversation to the user.

Tech stack: Python, FastAPI, LangChain, LangGraph, SQLAlchemy,
PostgreSQL, Qdrant, Redis, Next.js, React, TypeScript, Tailwind CSS,
Google Gemini (with Ollama supported for local development).

Building it taught me a lot about designing an efficient RAG pipeline,
managing conversation history without exceeding LLM context limits,
implementing streaming responses, reliable intent classification, and
scalable backend architecture. Planned future improvements include
multi-model support, tool calling, voice conversations, and image
understanding.
